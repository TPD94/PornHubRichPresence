
#### PornHubRichPresence - A Discord Rich Presence Client  
![Rich Presence Example](https://i.imgur.com/FOTu43u.png)
  
## Quick Start  

 
 **Chrome Steps**
 
 - Navigate to your chrome install in a command prompt (usually located in `C:\Program Files\Google\Chrome\Application`)
 - Force a new debugging session with `chrome.exe --remote-debugging-port=6000  --user-data-dir="C:\chrome-debug"`

**Python Steps**

 - Install requirements with `pip install -r requriements.txt`
 - Run the program `python main.py`
 
 ### Begin watching!
 Your rich presence should now update with whatever video you choose to watch! Now your friends can get some inspiration too! When finished your rich presence will automatically clear or you can simply stop the script.


**KNOWN LIMITATIONS**

    

 `- Can't detect active tab if more than one video is open in your browser`
